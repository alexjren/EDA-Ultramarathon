# Ultramarathon EDA

A personal exploratory data analysis (EDA) project focused on ultramarathon race data. This notebook investigates participation patterns, athlete demographics, and performance trends using Python-based data analysis tools.

The goal of this project is to practice real-world data cleaning, visualization, and insight generation while exploring an endurance sports dataset.

---

## Project Overview

This project performs exploratory data analysis on ultramarathon race data to uncover trends such as:

- Participation patterns  
- Gender distribution  
- Geographic comparisons  
- Performance characteristics  

The analysis emphasizes reproducible workflows, data visualization, and statistical results.

---

## Tech Stack

- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Key Analysis Areas

- Data cleaning and preprocessing  
- Handling missing and inconsistent values  
- Exploratory statistics  
- Visual trend analysis  
- Comparative demographic insights

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/alexjren/EDA-Ultramarathon.git
cd EDA-Ultramarathon
```

### 2. Install dependencies

```bash
pip install requirements.txt
```

### 3. Launch the notebook

```bash
jupyter notebook
```

Open the notebook file and run cells sequentially.
